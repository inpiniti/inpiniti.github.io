- branch 생성
    ```
    git checkout -b develop
    ```

- branch 확인
    ```
    git branch
    ```

- 푸시
    ```
    git push origin develop
    ```

- 코드 동기화
    ```
    git pull upstream develop
    ```
    
- 브런치 삭제
    ```
    git branch -d develop
    ```
