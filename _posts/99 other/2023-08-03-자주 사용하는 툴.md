# dev tool

- Android Studio

    - AOS 개발시 사용한다.
    
- VS Code

    - js 개발시 사용한다.
    
- WebStorm

    - js 개발 및 docker 배포용으로 사용한다. (이거 안쓰고 배포하는걸 터들해야 할것 같음)

- Jupyter Notebook

    - python 개발시 사용하는데, 그냥 vscode에서 해도 상관없는것 같음

- FileZila

    - 파일 전송시 사용

    - 모든 서버 접속 때 이게 가장 편함

    - mRemoteNG를 알기전까지 사용했었음

- Git Bash

    - 로컬에서 cli 창띄울때 사용
